## 📝 만드는 방법 (사용법)

### 1. 레포지토리 클론하기
먼저 요리 시작을 위해 **재료**를 준비하세요. 터미널에서 다음을 입력합니다:

```bash
git clone https://github.com/username/momo.git
cd momo
```

### 2. 설치 및 실행
이제 재료를 손질하고 불을 올려볼까요?
```bash
# 설치
npm install

# 실행
npm start
```

### 3. 맛을 보기 (테스트)

재료가 잘 섞였는지 한 번 확인해보세요.
혹시 맛이 이상하다면(에러 발생) 걱정하지 말고 저를 불러주세요! 함께 해결해 드릴게요.
